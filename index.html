弘揚儒釋道教育

一條龍教育理念

道德教育爲本

强調因果教育

教育宗旨理念

加拿大弟子規學會、奧羅拉國際學校聯合開辦弘毅私塾，將秉承上淨下空老和尚的一條龍教學理念和教學方法，以德行教育爲主，通過讀誦背誦《聖學根之根》、《四書》、《五經》培養學生定力，配合實踐習勞惜福，力行《弟子規》，培育弘揚中華傳統文化之聖賢人才！

招生條件

面試

  學生需經過面試合格，方可錄取。

背景

有家庭讀經背景或者讀經教育機構讀經背景。

理念

家長須認同學校教育理念，對學校和老師有足够的信任，積極配合在家落實學校教學理念。

家長

家長須熱愛傳統文化，願意參加定期家長共學，學習師長理念。主要學習內容《幸福人生講座-細講弟子規》、《做孩子一生的貴人》、《禮記·學記》、《傳統文化如何學習》、《太上感應篇講記》等。

環境

家庭無不良嗜好，願意爲學生創造清淨家庭環境，屏蔽電子産品及游戲。入學須簽訂《家長公約》，需踐行幷遵守，如不能遵守、且嚴重違反《家長公約》的家庭，本校有權勸退學生。

父母學校

父母參加父母學校，共同學習一條龍教育理念，實現家庭和學校的緊密配合。

幼兒養性，童蒙養正，少年養志，厚德載物

让教育回归“教人做人”的本质，通过一生的持续学习，实现身心和谐、社会和谐、世界和谐。

課程 設置

周末學習，定期講座，全日制學習

拜佛

Class time: 9:00am - 10am

早課淨業集資。

德行

Class time: 9:00am - 10am

《弟子規》、《德育故事》，以孝悌爲主，結合因果教育。

讀誦

Class time: 9:00am - 10am

讀誦《聖學根之根》，培養定力爲主。

學術

Class time: 9:00am - 10am

中英文識字習字，閲讀，寫作。

禮儀

Class time: 9:00am - 10am

《常禮舉要》、《一日常規》，將應對進退、待人接物等禮節落實生活實踐。

習勞

Class time: 9:00am - 10am

培養孩子生活能力和勤勞的態度。

樂教

Class time: 9:00am - 10am

經典吟誦、德音雅樂，如古典音樂熏陶、手語歌唱演等，陶冶情操，啓發孩童的善良之心。

體育

Class time: 9:00am - 10am

强身健體，鍛煉心志。
